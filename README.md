# shakacode

The official ShakaCode package on npm.

This package provides verified links to ShakaCode projects, npm packages, and
community resources. It is intentionally small and has no runtime dependencies.

## Usage

Run the CLI:

```sh
npx shakacode
```

Or install the package:

```sh
npm install shakacode
```

## Official Links

- Website: https://www.shakacode.com/
- GitHub: https://github.com/shakacode
- npm: https://www.npmjs.com/org/shakacode
- React on Rails: https://github.com/shakacode/react_on_rails
- Shakapacker: https://github.com/shakacode/shakapacker

## Security

This package has no dependencies and no install lifecycle scripts. If you find
a security issue in any ShakaCode package, please report it privately through
the appropriate GitHub repository security advisory flow.
